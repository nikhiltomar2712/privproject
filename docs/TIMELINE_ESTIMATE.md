# Timeline estimate (free/open approach)

These are realistic ranges assuming a small team (you + Ayush) and mostly open-source components.

## Best-case (you already have hardware + docs)
- 1–2 days: planning + MVP environment
- 3–7 days: chat MVP
- 5–14 days: RAG MVP
- 1–3 weeks: tools/workflows
- Total: **~3–5 weeks** for a usable assistant

## Typical case (hardware + setup + document ingestion)
- 1 week: planning + setup
- 1–2 weeks: chat MVP
- 2–4 weeks: RAG MVP
- 2–4 weeks: tools + evaluation
- Total: **~6–10 weeks**

## If you aim for fine-tuning

## What affects time the most
- amount/quality of documents
- GPU/compute availability
- how quickly you can evaluate and iterate

## Suggested weekly schedule

- Weekend: evaluation + documentation + improvements
